## About

CAD Dumps Quiz is an interactive web-based quiz application for practicing CAD-related questions. Test your knowledge with multiple quiz modes including random selections and sequential progression.

## Features

- 📋 Multiple quiz modes (Random, Sequence, Study)
- 🎯 Customizable question counts (10, 25, 60, 100, or all)
- ✅ Instant feedback on answers
- 📊 Real-time score tracking
- 📱 Progressive Web App (PWA) - works offline
- 🔒 Copy protection for content security
- 🎨 Clean, responsive UI

## Quiz Modes

- **Random**: Practice random sets of questions
- **Sequence**: Go through all questions in order
- **Study MCQs**: Browse all questions with answers and explanations

CAD Dumps Quiz App

This is an installable PWA app. It works offline after first opening.

How to use:
1. Open index.html in Chrome/Edge.
2. On mobile or desktop, use browser menu > Add to Home Screen / Install App.
3. After installing, open it like a normal app.

Modes included:
- Random 10Q
- Random 25Q
- Random 60Q
- Random 100Q
- Random All Questions
- Sequence
- Study MCQs scrolling mode

Quiz behavior:
- Select answer and click Next.
- App shows Correct/Wrong.
- If wrong, it shows the correct option.
- It waits 3 seconds and automatically moves to next question.
- Explanations are not shown after answering.
